Q . How to delete one commit or multiple commits permanently Write command for it.

-->

One can delete one or multiple commits by 
 'git reset --hard HEAD~n' 

such that, n represents number of commits!


This is a basic illustation of it:

jspnlp@kislayJosh:~/Documents/Assignment2/assi2/samarth$ git log -3
commit 8d164f620bccbbbeeeb7a99ae3c23347a3f5f8f3 (HEAD -> samarth)
Author: Samarth Kulkarni <svkulkarni23@gmail.com>
Date:   Mon Jan 15 14:16:30 2024 +0530

    un-necessary comment deleted-3

commit 8eab0d31cfa70c6cf4e708995f1d88d9f26966cc
Author: Samarth Kulkarni <svkulkarni23@gmail.com>
Date:   Mon Jan 15 14:15:59 2024 +0530

    un-necessary comment deleted-2

Author: Samarth Kulkarni <svkulkarni23@gmail.com>

    un-necessary comment deleted-1

jspnlp@kislayJosh:~/Documents/Assignment2/assi2/samarth$ git reset --hard HEAD~2
